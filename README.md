# moon — 문화유산 3D 체험 서비스

> 공공 3D 문화유산 데이터를 '다운로드용 파일'에서 '누구나 웹에서 즐기는 체험 콘텐츠'로.
> **2026 문화체육관광 AI·데이터 활용 공모전** / 문화데이터 활용 분야 — 우수사례 부문 출품작.

국립중앙박물관이 개방한 3D 소장품 데이터를 웹에서 바로 돌려보고, AI 도슨트의 해설을 듣고,
유물 연고지의 여행 정보까지 이어주는 문화유산 3D 체험 서비스입니다. (서비스명 확정 전 코드명 `moon`)

- **배포 URL**: _(배포 후 기입)_
- **활용 공공데이터**: 국립중앙박물관 소장품 3D · e뮤지엄 유물정보 API · 한국관광공사 TourAPI (모두 공공누리 제1유형)

## 주요 기능

- **유물 3D 뷰어** — 360° 회전·줌·풀스크린, **실제 크기 보기(사람 실루엣·눈금자 대조)**, 키보드 조작, WebGL 폴백
- **유물 카탈로그** — **208점**(3D 128 · 이미지 80), 즉시 검색 + 시대순·분류별 구분 + 재질·유형 필터
- **시대 타임라인** — 구석기~조선을 연대순으로, 시대 배경 + "생활 장면"(유물 쓰임새)과 함께 탐색
- **역사 이야기** — 스크롤에 따라 3D·이미지가 전환되는 스크롤리텔링(예: "흙에서 불로 — 그릇 1만 년")
- **멀티모달 AI** — AI 도슨트(LLM) + **온디바이스 의미 검색·비슷한 유물(임베딩)** + 유물 퀴즈 + 생성 이미지(재현)
- **AI 도슨트** — 유물 맥락을 주입받아 대화형으로 해설 (가드레일·"AI 생성" 고지)
- **관광지 연계** — 연고지 지도 + TourAPI 주변 관광지 (장애 시 스냅숏 폴백)
- **가상 전시관** — 분류별 구역의 3D 전시실을 키보드(WASD·화살표)로 걸으며 관람, 스포트라이트 연출, 유물 근접 Enter

## 정량 성과

국립중앙박물관 3D **128점**(공개 134점 중 다운로드 가능 전수)을 자체 파이프라인으로 경량화: **원본 약 6.1GB → 80.3MB (평균 97.3% 절감)**.
여기에 초기 시대(구석기~철기) **이미지 유물 80점**을 더해 총 **208점**. 모바일(Fast 4G)에서도 3D 인터랙션까지 수 초. 상세는 [`06-metrics.md`](./06-metrics.md).

## 기술 스택

Next.js (App Router) · TypeScript · React Three Fiber · Tailwind CSS · Vercel
3D 파이프라인: obj2gltf → gltf-transform (Draco 기하 압축 + WebP 텍스처)

## 로컬 실행

```bash
cd web
npm install
npm run build && npm run start   # http://localhost:3000
```

외부 API(도슨트·관광)는 `web/.env.local`의 키가 필요합니다. 키가 없으면 도슨트는 mock,
관광은 스냅숏으로 동작합니다. 에셋 파이프라인은 `cd web && npm run pipeline -- <slug>`.
