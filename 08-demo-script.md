# 08. 시연 영상 시나리오 (Demo Script)

> 작성일: 2026-06-14 · 용도: 우수사례 증빙 **시연 영상**(T-19) 촬영 대본. 배포 후 화면 녹화 + 내레이션.
> 목표 길이: **2분 30초~3분**. ⚠️ 양식이 영상 길이·형식을 지정하면 그에 맞춰 조정.
> 구성 원칙: 심사 배점(데이터활용·AI활용·완성도·활용성·사회적기여)을 **자연스러운 사용 흐름**으로 보여준다. 기능 나열이 아니라 "이용자 한 명의 체험"으로 엮는다.
> 촬영 전 준비: 배포 URL 동작 확인 · 시크릿 창(캐시 없는 첫 로딩으로 속도 어필) · 화면 1080p · 도슨트 실 LLM 연결 확인.

## 0. 오프닝 (0:00–0:15) — 문제 제기

| 화면 | 내레이션 |
|---|---|
| (선택) 박물관 3D 다운로드 페이지의 무거운 파일/전문 포맷을 잠깐 비춤 → 본 서비스 홈으로 전환 | "공공기관이 개방한 문화유산 3D 데이터, 하지만 전문 소프트웨어 없이는 열어볼 수 없었습니다. 저희는 이걸 누구나 웹에서 즐기는 체험으로 바꿨습니다." |
| 홈 히어로 "박물관 유리장 너머의 유물을, 지금 손끝에서" + "유물 14점 · 평균 97.5% 경량화" 지표 노출 | "국립중앙박물관 3D 데이터를, 설치 없이, 시공간 제약 없이." |

## 1. 3D 뷰어 — 완성도·데이터활용 (0:15–0:55)

| 화면 | 내레이션 |
|---|---|
| 카탈로그 진입 → **금동 반가사유상** 클릭 | "유물을 고르면," |
| 상세 페이지 3D 뷰어 로딩(진행률 잠깐) → 드래그로 360° 회전, 휠로 확대해 얼굴·손가락 디테일 | "이렇게 모든 각도에서, 실물처럼 자세히 살펴볼 수 있습니다. 원본 60MB를 1MB대로 압축해 모바일에서도 수 초면 열립니다." |
| 풀스크린·자동회전 버튼 잠깐 시연 | "수업 화면에 띄우거나, 가만히 감상하기에도 좋습니다." |
| 하단 메타데이터 + **공공누리 출처표시** 클로즈업 | "모든 유물에 시대·재질 정보와 국립중앙박물관 출처를 명기했습니다." |

## 2. AI 도슨트 — AI활용 (0:55–1:35)

| 화면 | 내레이션 |
|---|---|
| 같은 페이지의 AI 도슨트 영역으로 스크롤, 추천 질문 칩 "이 불상은 무슨 생각을 하고 있나요?" 클릭 | "궁금한 점은 AI 도슨트에게 물어봅니다." |
| 답변이 스트리밍으로 타이핑되는 모습 (유물 맥락 반영된 해설) | "이 AI는 해당 유물의 정보를 바탕으로, 박물관 도슨트처럼 쉽게 설명합니다." |
| 직접 질문 입력: "어디서 발견됐나요?" → 답변 | "자유롭게 대화하며 유물을 깊이 알아갈 수 있고," |
| "AI 생성 답변" 고지 라벨 클로즈업 | "답변에는 AI 생성 표시를 두어 신뢰성도 챙겼습니다." |

## 3. 관광지 연계 — 데이터활용·활용성 (1:35–2:05)

| 화면 | 내레이션 |
|---|---|
| 같은 페이지 하단 "직접 가서 만나기" 지도 + 주변 관광지 카드(이미지·거리) | "감상에서 끝나지 않습니다. 한국관광공사 데이터를 실시간으로 연동해," |
| 지도 마커·주변 관광지 목록 훑기 | "이 유물을 직접 만날 수 있는 곳과 주변 여행지까지 이어줍니다. 사전 체험에서 실제 방문으로." |

## 4. 가상 전시관 — 혁신성·독창성 (2:05–2:35)

| 화면 | 내레이션 |
|---|---|
| 상단 내비 "가상 전시관" → 입장 화면(용량 안내) → 입장 | "여러 유물을 한 번에 거닐고 싶다면, 가상 전시관으로." |
| '시대를 걷다' 전시실에서 시점 이동, 신석기→조선 6점 시대순 관람 | "신석기 토기부터 조선 백자까지, 시대순으로 배치된 3D 공간을 둘러봅니다." |
| 유물 하나 클릭 → 상세로 이동 | "마음에 드는 유물은 클릭해 자세히." |

## 촬영 체크리스트
- [ ] 배포 URL에서 전 흐름 1회 리허설 (도슨트 실 LLM 응답 확인)
- [ ] 시크릿 창·캐시 비움 상태로 로딩 속도 자연스럽게 노출
- [ ] 3D 회전은 천천히(어지럽지 않게), 자동회전 과용 금지
- [ ] 출처표시·"AI 생성" 라벨은 반드시 한 번씩 클로즈업 (심사 정직성·필수 요건)
- [ ] 자막 권장(무음 시청 대비), 핵심 수치(97.5% 경량화 등)는 자막으로 강조
- [ ] 영상 끝 또는 설명란에 데이터 출처(국립중앙박물관·한국관광공사) 한 줄 명기
